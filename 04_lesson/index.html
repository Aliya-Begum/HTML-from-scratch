HTML Lists

Now we are going to discuss about HTML Lists. There are three types of Lists HTML supports

Ordered List

  1. list-item-one
  2. list-item-two
  3. list-item-three

We can also use Ordered list as reversed Ordered list

  1. list-item-one
  2. list-item-two
  3. list-item-three

Unordered List

  • list-item-one
  • list-item-two
  • list-item-three
